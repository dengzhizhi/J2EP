Changelog for jEasy Extensible Proxy 1.0

Changes listed are only major new features and doesn't include bug fixes. To monitor bugs please check the bug tracker.

  • Added check of the Via header so that cyclic requests aren't allowed. Cyclic requests are requests from the proxy that are returning to the proxy once again, creating a never ending loop.
  • More proofreading of the manual and on some parts a bit of rewriting. All in all the manual should be easier to read and contain fewer spelling errors.
